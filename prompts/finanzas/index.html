#1 — Narrativa de Reporte Financiero
Finanzas
▾
Actúa como un controller financiero que presenta resultados al consejo directivo. Los datos clave son: Ingresos: [cifra] (vs presupuesto: [cifra]). Costos: [cifra]. Margen bruto: [%]. Las variaciones más relevantes son: [describe]. Escríbeme un resumen ejecutivo de máximo 300 palabras con: 1) El headline, 2) Variaciones clave, 3) Acciones correctivas, 4) Perspectiva para el siguiente periodo. Sin jerga contable innecesaria.
🧮
#2 — Justificación de Presupuesto
Finanzas
▾
Actúa como un director de finanzas. El área de [área] solicita un presupuesto de [cantidad] para [proyecto]. Los beneficios esperados son: [lista]. El costo de NO hacerlo sería: [describe]. Escríbeme una justificación de una página con: problema actual, solución propuesta, inversión requerida, ROI esperado, y riesgos de no aprobar. Formato ejecutivo con números claros.
📉
#3 — Análisis de Variaciones vs Presupuesto
Finanzas
▾
Actúa como un analista financiero. Tengo las siguientes variaciones entre real y presupuesto: [pega los datos o describe las variaciones principales por rubro]. Necesito: 1) Explicación en lenguaje ejecutivo de cada variación relevante, 2) Clasificar si es favorable o desfavorable y por qué, 3) Identificar las 3 variaciones que más impactan el resultado final, 4) Redactar el párrafo de "causas principales" para el reporte mensual a dirección.
🎯
#4 — KPIs Financieros para mi Área
Finanzas
▾
Actúa como un consultor de finanzas corporativas. Soy [tu rol] en una empresa de [industria, tamaño]. Necesito definir los 5-7 KPIs financieros más relevantes para monitorear el desempeño de [área o proceso que manejas]. Para cada KPI dame: nombre, definición, cómo calcularlo, frecuencia de medición, y un rango de referencia de "bueno / regular / malo" para mi industria.
