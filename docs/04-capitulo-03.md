# CAPÍTULO 3: El Arte de Pedir (Prompts para Gente Normal)

> **Estado:** 📋 Resumen listo — pendiente de escritura completa

## Objetivo
Enseñar a escribir prompts efectivos sin usar jerga técnica, con el framework ROCA.

## Contenido planificado

El capítulo más importante del libro en términos de habilidad. Aquí se introduce el framework ROCA (Rol + Objetivo + Contexto + Acción), que es una fórmula sencilla para escribir prompts que realmente funcionen.

### Subtemas clave:
1. **Por qué "preguntarle bien" a la IA es tu nueva superpotencia**
2. **El framework ROCA: Rol + Objetivo + Contexto + Acción** — Con ejemplos paso a paso
3. **10 prompts listos para copiar y pegar** — Correos, reportes, resúmenes, análisis
4. **Cómo iterar** — La conversación con la IA no es de un solo mensaje
5. **Prompts que nunca funcionan (y por qué)** — Antes vs. después
