## Apéndice A: Banco de 30 Prompts por Profesión

Estos prompts complementan los 10 Prompts Mágicos del Capítulo 3 y los ejemplos del Capítulo 4. Todos usan el framework ROCA y están listos para copiar desde el sitio web companion. Adapta los campos entre `[corchetes]` a tu situación.

### Marketing y Ventas (5 prompts)

**A1. Análisis de competencia exprés**
```
Actúa como un analista de inteligencia competitiva. Voy a 
darte información pública sobre mi competidor [nombre]. 
Analiza: 1) Su propuesta de valor principal, 2) A qué 
público le habla, 3) Qué están haciendo bien que yo 
debería considerar, 4) Qué oportunidades están dejando 
que yo podría aprovechar. Información: [pega textos de 
su sitio web, redes o materiales públicos]
```

**A2. Script para llamada en frío**
```
Actúa como un trainer de ventas. Necesito un script para 
una llamada en frío de máximo 90 segundos. Vendo [producto/
servicio] a [tipo de empresa]. El dolor principal de mi 
cliente es [describe]. Mi diferenciador es [describe]. 
Dame el script con: apertura que enganche (sin "¿cómo 
está?"), pregunta de dolor, propuesta en una oración, y 
cierre para agendar una reunión. Incluye 3 respuestas 
para objeciones comunes.
```

**A3. Análisis de métricas de campaña**
```
Soy gerente de marketing digital. Estos son los resultados 
de mi última campaña en [plataforma]: [pega métricas: 
impresiones, clics, CTR, conversiones, costo, etc.]. 
Mi objetivo era [describe]. Mi presupuesto fue [monto]. 
Dime: 1) ¿La campaña fue exitosa? ¿Por qué sí o no?, 
2) Qué métricas están por debajo del benchmark de mi 
industria, 3) 3 ajustes concretos para mejorar en la 
siguiente campaña.
```

**A4. Copy para email marketing**
```
Actúa como un copywriter especializado en email marketing. 
Necesito un correo para [objetivo: lanzamiento / promoción / 
reactivación de clientes inactivos / newsletter]. Mi marca 
es [describe]. El público es [describe]. El tono es [describe]. 
Dame: asunto del correo (3 opciones), preview text, cuerpo 
del correo (máximo 200 palabras) y CTA. Que no suene a 
spam y que se sienta personal.
```

**A5. Estrategia de precios**
```
Actúa como un consultor de pricing. Mi producto/servicio 
es [describe]. Mis costos son aproximadamente [describe]. 
Mi competencia cobra entre [rango]. Mi público objetivo 
es [describe]. Ayúdame a: 1) Definir 3 posibles estrategias 
de precio (penetración, premium, basada en valor), 
2) Pros y contras de cada una para mi caso, 3) Cómo 
comunicar el precio al cliente sin que se enfoque solo 
en el número.
```

### Recursos Humanos (5 prompts)

**A6. Plan de onboarding**
```
Actúa como un especialista en experiencia del empleado. 
Necesito un plan de onboarding para un nuevo [puesto] en 
una empresa de [industria, tamaño]. El onboarding debe 
cubrir sus primeros 30 días. Incluye: semana 1 (orientación 
y cultura), semana 2 (capacitación técnica), semana 3 
(integración al equipo), semana 4 (primeros entregables). 
Para cada semana dame actividades específicas, responsable 
y objetivo. Formato de tabla.
```

**A7. Encuesta de clima laboral**
```
Actúa como un consultor de desarrollo organizacional. 
Necesito diseñar una encuesta de clima laboral para una 
empresa de [tamaño] empleados en [industria]. Las áreas 
que más nos preocupan son: [liderazgo / comunicación / 
compensación / desarrollo profesional / balance vida-trabajo]. 
Dame 20 preguntas usando escala Likert (1-5), organizadas 
por categoría, más 3 preguntas abiertas. Que el lenguaje 
sea claro y no induzca la respuesta.
```

**A8. Guión para conversación difícil**
```
Actúa como un coach de liderazgo. Necesito tener una 
conversación difícil con un colaborador sobre [situación: 
bajo desempeño / actitud / puntualidad / conflicto con 
compañero]. Contexto: [describe la situación, historial, 
intentos previos]. Quiero ser directo pero respetuoso. 
Dame: 1) Cómo abrir la conversación, 2) Puntos clave 
a tocar, 3) Cómo manejar si se pone defensivo, 4) Cómo 
cerrar con acuerdos claros. Máximo una página.
```

**A9. Descripción de puesto interno**
```
Soy de RRHH y necesito documentar el puesto de [nombre]. 
El ocupante actual es [nombre/rol] y reporta a [quién]. 
Sus responsabilidades principales son: [lista general]. 
Las herramientas que usa son: [lista]. Escríbeme una 
descripción de puesto formal con: objetivo del puesto 
(1 párrafo), responsabilidades clave (6-8), requisitos 
de perfil, competencias necesarias, y relaciones internas 
(con qué áreas interactúa). Formato profesional.
```

**A10. Política interna**
```
Actúa como un consultor de RRHH. Necesito redactar una 
política de [tema: home office / código de vestimenta / 
uso de IA / redes sociales / viáticos]. La empresa tiene 
[tamaño] empleados en [industria]. Los puntos que debe 
cubrir son: [lista de temas]. El tono debe ser claro 
y directo, sin jerga legal innecesaria. Incluye: objetivo 
de la política, alcance, lineamientos, excepciones y 
consecuencias. Máximo 2 páginas.
```

### Finanzas y Contabilidad (5 prompts)

**A11. Análisis de flujo de efectivo**
```
Actúa como un analista financiero. Te voy a dar los datos 
de mi flujo de efectivo de los últimos [período]. Necesito 
que identifiques: 1) Meses con mayor presión de liquidez, 
2) Patrones estacionales, 3) Gastos que crecieron más que 
los ingresos, 4) Recomendaciones para mejorar el flujo 
en los próximos 3 meses. Presenta las conclusiones para 
alguien que no es financiero. Datos: [pega los datos]
```

**A12. Presupuesto departamental**
```
Soy [tu puesto] y necesito armar el presupuesto de mi 
área para [período]. El presupuesto del año pasado fue 
[monto] distribuido así: [categorías y montos]. Este año 
espero [cambios: crecimiento del equipo, nuevos proyectos, 
recortes]. Dame: 1) Una estructura de presupuesto en 
formato de tabla, 2) Cómo justificar cada línea ante 
finanzas, 3) Dónde podría haber espacio para negociar 
más presupuesto, 4) Gastos que podría recortar sin 
impacto operativo.
```

**A13. Explicación de estados financieros**
```
Actúa como un director de finanzas que sabe comunicar 
con claridad. Voy a darte los números clave de nuestro 
[estado de resultados / balance general]. Necesito que 
escribas una explicación de máximo 300 palabras dirigida 
a [gerentes de área / equipo comercial / consejo] que 
responda: ¿cómo nos fue?, ¿por qué?, ¿qué sigue? Usa 
analogías simples si es necesario. Datos: [pega los 
números clave]
```

**A14. Checklist de cierre mensual**
```
Actúa como un controller financiero. Necesito crear un 
checklist de cierre mensual para el área de contabilidad 
de una empresa [tamaño] en [industria] en México. Incluye: 
actividades previas al cierre, actividades del cierre, 
conciliaciones necesarias, revisiones de control, y 
entregables finales con fecha límite relativa (ej: "día 
3 hábil del mes"). Formato de tabla con: actividad, 
responsable, fecha, y estatus.
```

**A15. Análisis de rentabilidad por producto**
```
Tengo [X] productos/servicios con los siguientes datos 
de ingresos y costos: [pega datos]. Necesito un análisis 
de rentabilidad que incluya: 1) Margen bruto y margen 
neto por producto, 2) Ranking de más a menos rentable, 
3) Productos que están por debajo del margen mínimo 
aceptable de [X%], 4) Recomendaciones: ¿cuáles 
potenciar, cuáles replantear, cuáles descontinuar? 
Preséntalo en tabla y con un párrafo de conclusión.
```

### Legal y Compliance (5 prompts)

**A16. Comparativa de cláusulas**
```
Actúa como un abogado corporativo. Te voy a dar dos 
versiones de [tipo de cláusula: confidencialidad / 
penalización / terminación anticipada] de un contrato. 
Compáralas e identifica: 1) Diferencias clave entre 
ambas versiones, 2) Cuál favorece más a cada parte, 
3) Riesgos de cada versión, 4) Sugerencia de redacción 
que equilibre los intereses. Versión A: [pega]. 
Versión B: [pega]. NOTA: Verificaré todo antes de usar.
```

**A17. Resumen de nueva regulación**
```
Actúa como un abogado especializado en [área: fiscal / 
laboral / protección de datos / comercio exterior] en 
México. Te voy a pegar el texto de una nueva disposición 
regulatoria. Necesito: 1) Resumen en 5 puntos clave, 
2) ¿A quién le aplica?, 3) ¿Qué tenemos que hacer para 
cumplir?, 4) Fechas límite importantes, 5) Sanciones 
por incumplimiento. Lenguaje claro, sin jerga legal 
innecesaria. Texto: [pega]. NOTA: Validaré contra 
fuente oficial.
```

**A18. Aviso de privacidad**
```
Necesito redactar un aviso de privacidad simplificado 
para [tipo de empresa] en México, conforme a la LFPDPPP. 
Los datos que recabamos son: [lista]. Los usamos para: 
[finalidades]. [Sí/No] compartimos datos con terceros. 
El responsable es [nombre/empresa]. Dame un borrador 
que sea legalmente sólido pero entendible para el usuario 
promedio. Máximo una página. NOTA: Lo revisaré con mi 
abogado antes de publicar.
```

**A19. Acta de reunión con valor legal**
```
Actúa como un secretario del consejo de administración. 
Necesito redactar un acta de la reunión de [tipo: consejo / 
asamblea / comité] celebrada el [fecha]. Asistieron: 
[nombres y cargos]. Los puntos tratados fueron: [lista]. 
Los acuerdos fueron: [lista]. Dame el acta en formato 
formal con: encabezado, lista de asistentes, orden del 
día, desarrollo, acuerdos numerados, y cierre con espacio 
para firmas.
```

**A20. Términos y condiciones**
```
Necesito un borrador de términos y condiciones para 
[tipo de servicio/producto/plataforma]. Mi empresa es 
[describe]. El servicio consiste en [describe]. Los 
temas que debo cubrir son: uso aceptable, propiedad 
intelectual, limitación de responsabilidad, política 
de cancelación/devolución, jurisdicción. Que sea claro 
y razonablemente breve. Jurisdicción: México. NOTA: 
Revisión legal obligatoria antes de publicar.
```

### Administración y Gerencia (5 prompts)

**A21. Evaluación de desempeño**
```
Actúa como un gerente preparando evaluaciones de 
desempeño. El colaborador es [nombre/puesto]. Sus 
objetivos del periodo eran: [lista]. Sus resultados 
fueron: [describe]. Sus fortalezas observadas: [lista]. 
Áreas de mejora: [lista]. Escríbeme: 1) Evaluación 
narrativa de un párrafo (equilibrada, no solo positiva), 
2) Calificación sugerida por objetivo, 3) Plan de 
desarrollo para el siguiente periodo con 3 acciones 
específicas. Tono: constructivo y directo.
```

**A22. Comunicado organizacional**
```
Necesito redactar un comunicado interno para anunciar 
[cambio: reestructura / nuevo beneficio / cambio de 
política / resultado importante]. La audiencia es 
[toda la empresa / un área / gerentes]. El tono debe 
ser [tranquilizador / celebratorio / informativo / 
serio]. Puntos que debe cubrir: [lista]. Que no suene 
a corporativismo vacío — que sea directo y humano. 
Máximo 250 palabras. Dame 2 versiones.
```

**A23. Agenda de junta efectiva**
```
Necesito estructurar una junta de [duración] minutos 
sobre [tema principal]. Los participantes son [describe]. 
Los temas a cubrir son: [lista con prioridad]. Las 
decisiones que necesitamos tomar son: [lista]. Dame 
una agenda con: tiempo asignado por tema, responsable 
de presentar cada punto, espacio para decisiones, y 
5 minutos al final para pendientes. Incluye una 
pregunta de apertura que enfoque la conversación.
```

**A24. Dashboard de KPIs**

```
Soy gerente de [área] y necesito definir los KPIs para 
mi tablero de seguimiento mensual. Mi equipo tiene 
[tamaño] personas. Nuestros objetivos principales son: 
[lista]. Las métricas que actualmente reviso son: [lista]. 
Ayúdame a: 1) Validar si mis KPIs actuales son los 
correctos, 2) Sugerir KPIs que me faltan, 3) Definir 
meta y frecuencia de medición para cada uno, 4) Diseñar 
la estructura del dashboard (qué va arriba, qué abajo). 
Formato de tabla.
```

**A25. Propuesta de mejora de proceso**
```
Actúa como un consultor de eficiencia operativa. Hay un 
proceso en mi área que no funciona bien: [describe el 
proceso actual paso a paso, quién interviene, cuánto 
tarda, dónde se atora]. Los problemas principales son: 
[lista]. Necesito: 1) Diagnóstico de las causas raíz, 
2) Proceso rediseñado paso a paso, 3) Qué pasos se

en tiempo/costo. Preséntalo como si fuera una propuesta
```

```
Mi negocio es [describe en 2-3 oraciones]. Necesito 
3 versiones de mi elevator pitch (máximo 30 segundos 
cada uno) adaptadas a: 1) Un posible inversionista 
(enfoque: oportunidad y retorno), 2) Un posible cliente 
(enfoque: qué problema le resuelvo), 3) Un posible 
socio estratégico (enfoque: qué ganamos juntos). Que
```

**A29. Plan financiero para emprendedor**

```
1) Punto de equilibrio (cuántas unidades/clientes
```

**A30. Automatización de atención al cliente**

```
predefinidas para cada pregunta frecuente (tono cercano,

responde Y", 3) Identificar en qué punto debe intervenir 
un humano, 4) Sugerir una herramienta gratuita o
```

## Apéndice B: Recursos Gratuitos Recomendados

### Cursos en español

- **Google: Fundamentos de IA Generativa** — Curso gratuito en Google Cloud Skills Boost. Corto, claro y sin código. Buen punto de partida.
- **LinkedIn Learning: cursos de IA para profesionales** — Si tu empresa tiene suscripción, busca "IA generativa" o "ChatGPT". Hay varios cursos cortos en español.
- **Platzi: curso de ChatGPT** — Platzi tiene cursos accesibles sobre prompts e IA aplicada. Algunos gratuitos, otros con suscripción.
- **Coursera: IA para todos (Andrew Ng)** — Disponible en español con subtítulos. No es técnico — está diseñado para profesionales y gerentes.

### Canales de YouTube en español

- **DotCSV** — Uno de los canales más populares sobre IA en español. Mezcla noticias con explicaciones claras.
- **La IA al Día** — Novedades de IA explicadas para público general.
- **Carlos Muñoz AI** — Tutoriales prácticos de herramientas de IA aplicadas al trabajo.

### Newsletters

- **The Rundown AI** — Newsletter diaria con las novedades de IA más relevantes (en inglés, pero corta y directa).
- **Ben's Bites** — Similar, con enfoque en herramientas prácticas (en inglés).
- **Suscríbete a los blogs oficiales** de OpenAI, Anthropic y Google DeepMind para enterarte de actualizaciones directamente.

### Comunidades

- **r/ChatGPT y r/artificial en Reddit** — Comunidades activas donde la gente comparte prompts, trucos y casos de uso.
- **Grupos de LinkedIn sobre IA en LATAM** — Busca "IA para profesionales LATAM" o "Inteligencia Artificial México". Hay varios grupos activos.
- **Discord y Slack** — Muchas herramientas de IA tienen comunidades oficiales donde puedes hacer preguntas y aprender de otros usuarios.

### Herramientas para practicar (gratuitas)

- **ChatGPT** (chat.openai.com) — Versión gratuita disponible.
- **Claude** (claude.ai) — Versión gratuita disponible.
- **Gemini** (gemini.google.com) — Gratuito con cuenta de Google.
- **Microsoft Copilot** (copilot.microsoft.com) — Versión gratuita con funcionalidad limitada.

## Apéndice C: Comparativa de Herramientas de IA

| Característica | ChatGPT | Claude | Gemini | Microsoft Copilot |
|---|---|---|---|---|
| **Empresa** | OpenAI | Anthropic | Google | Microsoft |
| **Versión gratuita** | Sí | Sí | Sí | Sí |
| **Precio versión premium** | ~$20 USD/mes | ~$20 USD/mes | ~$20 USD/mes | ~$20-30 USD/mes |
| **Fortaleza principal** | Versatilidad, comunidad grande | Textos largos, seguir instrucciones complejas | Integración con Google, info actualizada | Integración con Office 365 |
| **Mejor para** | Uso general, escritura, código | Análisis detallados, documentos extensos | Quien vive en el ecosistema Google | Quien vive en el ecosistema Microsoft |
| **Integración nativa** | Plugins, GPTs personalizados | Proyectos, artefactos | Gmail, Docs, Sheets, Drive | Word, Excel, PowerPoint, Outlook |
| **Limitación principal** | Puede ser impreciso con datos recientes | Comunidad más pequeña | A veces menos profundo en análisis | Requiere licencia M365 para máximo provecho |
| **Disponible en español** | Sí | Sí | Sí | Sí |
| **Mejor punto de entrada** | chat.openai.com | claude.ai | gemini.google.com | copilot.microsoft.com |

**Recomendación:** no te cases con una sola herramienta. Empieza con la que más te acomode, pero prueba al menos dos. Cada una tiene personalidad diferente y para ciertos tipos de tareas vas a preferir una sobre otra. Lo importante es dominar el framework ROCA — eso funciona en todas.

## Apéndice D: Glosario Extendido

Los 15 términos del Capítulo 1 más 15 términos adicionales que te van a servir conforme avances.

### Términos del Capítulo 1 (referencia rápida)

| # | Término | Definición corta |
|---|---------|-----------------|
| 1 | Prompt | Tu instrucción o pregunta a la IA |
| 2 | IA Generativa | IA que crea contenido nuevo |
| 3 | LLM | "Cerebro" detrás de ChatGPT, Claude, etc. |
| 4 | Alucinación | Cuando la IA inventa datos falsos |
| 5 | Token | Unidad de medida de texto para la IA |
| 6 | Contexto | Información que la IA "recuerda" en la conversación |
| 7 | Modelo | Versión específica de IA (ej. GPT-4) |
| 8 | Fine-tuning | Personalizar un modelo con datos propios |
| 9 | API | Conexión programática entre sistemas |
| 10 | Automatización | Tareas que se ejecutan solas |
| 11 | No-code | Crear sin programar (Zapier, Make, etc.) |
| 12 | Chatbot | Programa que conversa contigo |
| 13 | Machine Learning | IA que aprende de datos |
| 14 | Sesgo (Bias) | Prejuicios en la IA por sus datos de entrenamiento |
| 15 | Copilot | IA integrada en tus herramientas de trabajo |

### 15 términos adicionales

**16. Agente (Agent):** Una IA que puede ejecutar secuencias de tareas de forma autónoma — no solo responde, sino que actúa. Por ejemplo, investigar un tema, escribir un resumen y mandarlo por correo, todo sin intervención paso a paso.

**17. RAG (Retrieval-Augmented Generation):** Técnica donde la IA busca información en documentos específicos antes de responder. Permite que la IA "consulte" tus archivos internos en lugar de depender solo de su entrenamiento general.

**18. Temperatura:** Parámetro que controla qué tan creativa o conservadora es la IA. Temperatura baja = respuestas más predecibles y precisas. Temperatura alta = respuestas más variadas y creativas.

**19. Few-shot prompting:** Darle a la IA 2-3 ejemplos del resultado que quieres antes de pedirle que genere uno nuevo. Mejora la calidad dramáticamente. Ejemplo: "Aquí van 3 correos que me gustan. Escribe uno nuevo con ese mismo estilo."

**20. Zero-shot prompting:** Pedirle a la IA que haga algo sin darle ejemplos previos. Es lo que haces normalmente cuando le escribes un prompt directo.

**21. Chain of thought (Cadena de pensamiento):** Pedirle a la IA que piense paso a paso antes de dar su respuesta final. Útil para problemas complejos. Se activa con frases como "piensa paso a paso" o "razona antes de responder".

**22. Multimodal:** IA que puede procesar y generar diferentes tipos de contenido: texto, imágenes, audio, video. Las versiones más recientes de las herramientas principales ya son multimodales.

**23. Embedding:** Representación numérica de texto que la IA usa internamente. No necesitas entenderlo a fondo, pero aparece cuando se habla de búsquedas semánticas o bases de datos de conocimiento.

**24. Hallucination rate:** Frecuencia con la que un modelo inventa información. Varía entre modelos y tipos de pregunta. Las preguntas factuales específicas tienen mayor riesgo de alucinación que las tareas creativas.

**25. System prompt:** Instrucciones iniciales que definen cómo debe comportarse la IA antes de que tú le hables. Es lo que usan las empresas para configurar chatbots personalizados. Cuando tú defines un "Rol" con ROCA, estás haciendo algo similar.

**26. Workflow (Flujo de trabajo):** Secuencia de pasos automatizados que conectan varias herramientas. En Zapier se llaman "Zaps", en Make "Escenarios", en Power Automate "Flujos".

**27. Playground:** Interfaz de prueba que ofrecen las empresas de IA donde puedes experimentar con diferentes configuraciones (temperatura, modelo, longitud de respuesta). Útil para usuarios avanzados.

**28. GPT personalizado / Custom GPT:** Versión de ChatGPT configurada para una tarea específica con instrucciones, documentos y personalidad predefinidos. Cualquier usuario de ChatGPT Plus puede crear uno.

**29. Límite de contexto (Context window):** Cantidad máxima de texto que la IA puede procesar en una conversación. Si superas el límite, la IA "olvida" el inicio. Los modelos más recientes manejan textos cada vez más largos.

**30. Guardrails:** Reglas y filtros que se le ponen a la IA para evitar que genere contenido dañino, incorrecto o fuera de las políticas de la empresa. Importantes cuando se implementa IA en productos para clientes.
